# Test Unitaire App

Bienvenue dans ce projet de démonstration pour les tests unitaires et d'intégration avec React.
L'objectif est d'illustrer les bonnes pratiques en matière de validation de formulaire et d'automatisation (CI/CD).

## 🚀 Fonctionnalités
- Formulaire d'inscription avec validation poussée
- Tests Unitaires (UT) couvrant les fonctions utilitaires et les composants (Coverage: 100%)
- Tests d'Intégration (IT) simulant les parcours utilisateurs
- Déploiement automatisé sur GitHub Pages et NPM via GitHub Actions
- Rapport de couverture sur Codecov

## 🛠️ Installation

1. Clonez ce dépôt :
   ```bash
   git clone https://github.com/ibra-sko/Test-Unitaire.git
   ```
2. Installez les dépendances :
   ```bash
   npm install
   ```

## 🧪 Lancer les Tests

Pour exécuter les tests (UT & IT) et générer le rapport de couverture localement :
```bash
npm run test
```
Le rapport de couverture se trouvera dans le dossier `coverage/lcov-report`. Ouvrez `index.html` dans votre navigateur.

## 📦 Lancer le Projet Localement

```bash
npm run start
```
L'application sera accessible sur [http://localhost:3000](http://localhost:3000).
